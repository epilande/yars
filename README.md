## Quick Start

#### Clone this repo

```bash
$ git clone https://github.com/epilande/yars.git
$ cd yars
```

#### Install dependencies

```bash
$ npm install
```

#### Launch dev environment

```bash
$ npm run dev
```

#### Launch prod environment

```bash
$ npm run prod
```
